KCMT-SIS: Wish Email API

Send personalized wish emails for birthdays and anniversaries using this API.

Endpoint: /duggu-api/send-wish-email

Method: POST

Payload:

{
  "email": "example@example.com",
  "name": "John Doe",
  "designation": "Software Engineer",
  "dateOfBirth": "1990-01-01",
  "dateOfAnniversary": "2015-06-15"
}
